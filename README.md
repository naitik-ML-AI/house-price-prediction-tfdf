# house-price-prediction-tfdf
About This Project

This project builds a machine learning model to predict house prices using the Kaggle House Prices dataset.
A Random Forest model from TensorFlow Decision Forests (TF-DF) is used to analyze housing features and estimate sale prices.

The project demonstrates how decision forest algorithms can effectively handle tabular datasets with numerical and categorical features.

Technologies Used

Python

TensorFlow

TensorFlow Decision Forests

Pandas

NumPy

Matplotlib

Seaborn

Dataset

Dataset used: Kaggle House Prices – Advanced Regression Techniques

Features include:

Area

Number of rooms

Construction year

Garage size

Neighborhood

Many other housing attributes

The model predicts the SalePrice of a house.

Model Used

Random Forest Regression using TensorFlow Decision Forests

Key characteristics:

Handles categorical and numerical data

Robust against overfitting

Requires minimal preprocessing

Project Workflow

Load the dataset

Explore and visualize data

Split data into training and validation sets

Convert dataset to TensorFlow format

Train Random Forest model

Evaluate model performance

Analyze feature importance

Generate predictions for test dataset

Output

The model predicts house prices and generates a submission.csv file containing:

House ID

Predicted SalePrice
